Adminstration System


Admin guide:

  • To create a NEW name,
    1. Fill in the form and
    2. Click 'Add New name'
  • To EDIT/MAKE CHANGES to an exisitng name,
    1. Key in Name
    2. Click "Retrieve Data",
    3. Edit the relavant fields and then
    4. Click "Save"
  • To DELETE an exisitng pet
    1. Key in Name
    2. click "Retrieve Data" (This is to check the correct pet)
    3. Click "Delete"
  • To make *REMOVE ALL name
    1. click "Remove all records"
